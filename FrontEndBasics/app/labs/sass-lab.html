SASS LAB

This lab will turn this website into an ABB styled website.

Theme Rules

Colors

  1. Open _colors.scss
  2. Add the following colors and give them a name that describes the color. Copy syntax from previous colors.
    • #004b7a;
    • #0076b7
    • #ff000f
    • #6e6e6e
    • #ffff00

Typography

  1. Open _typography.scss
  2. Add the following @mixin's for italic and bold styles
    • Helvetica-Neue-Light-Italic
      • font-family: Helvetica Neue Light, Helvetica Neue eText Pro Light, Arial, sans-serif;
      • font-weight:300;
      • font-style: italic;
    • Helvetica-Neue-Light-Bold
      • font-family: Helvetica Neue Light, Helvetica Neue eText Pro Light, Arial, sans-serif;
      • font-weight:600;
      • font-style: normal;

Base

  1. Open _base.scss
  2. Add styles for strong and em styles
    • strong { @include Helvetica-Neue-Light-Bold; }
    • em { @include Helvetica-Neue-Light-Italic; }
  3. Starting with h1 styles step down to h6 change font-size from largest to smallest
    • h1 { font-size:$font-size-xxx-large; color:$dark-blue}
    • h2 { font-size:$font-size-xx-large; color:$dark-blue}
    • h3 { font-size:$font-size-x-large; }
    • h4 { font-size:$font-size-large; }
    • h5 { font-size:$font-size-regular; }
    • h6 { font-size:$font-size-small; }
  4. Keep your styles dry by combining like styles.
    • Remove {color:$dark-blue; } from h1, h2
    • Create new selector h1, h2 with style {color:$dark-blue; }

Modules

  1. Open _nav.scss
  2. Add styles to fix the navbar to match ABB's style guide
    • Add bottom border color
      • .navbar-default .navbar-nav > li { border-bottom: 3px solid transparent; &:hover { border-bottom: 3px solid $blue-gray; } }
    • Change navigation font color on hover
      • .navbar-default .navbar-nav > li > a:hover, .dropdown-menu > li > a:hover { color:$blue-gray; }
