CSS Selectors

If you learn nothing else today about CSS, try to remember these 3 things

  • Keep it Separate
  • Embrace the cascade
  • Keep it DRY (or Don't Repeat Yourself)

CSS Selectors (Basic)

  • Element Selectors - Least Specific - <Body>, <p>
  • Class Selectors - In-between - <.header>
  • Attribute Selectors - In-between - <Input[type=text]>
  • Id Selectors - Most Specific - <#header>

CSS Selectors (Advanced)

  • Element, Element div, p - Selects all <div> and <p> elements
  • Element Element div p - Selects all <p> inside <div> elements
  • Element > Element div > p - Selects all <p> that are direct children of a <div>
  • Element + Element div + p - Selects all <p> that are immediately next to a <div>
  • Element ~ Element p ~ ul - Selects all <ul> that are immediately next to a <p>

CSS Selectors Rules (Specificity)

  • The more specific the selector the higher precedence it takes
  • Specificity goes in order in id, class, attribute, element
  • If specificity # is the same, last written style takes precendence
  • !important tag will override specificity UNLESS there is another !important.
