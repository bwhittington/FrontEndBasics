CSS Positioning

  • Position Property
  • Static - Positions the element where it would normally appear, the default setting
  • Relative – Offsets the element from its normal position. Use top and left to set the distance positive or negative from the previous element in the DOM
  • Absolute - The element is removed from the HTML flow and positioned within the first non-static element. If all parents are static then it is placed in accordance to the browser window
  • Fixed - The element is positioned with absolute values, but in relation to the browser window
